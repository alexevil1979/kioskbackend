# Вопросы (Т-Банк first)

## Обязательно для старта

- [ ] **Какой вариант Т-Банка выбираем?**
  - [ ] **aQsi 6** (оплата + чек в одном устройстве)
  - [ ] **PAX + HS-K33 без УМКА** (`tbank_pos_printer`) + CloudKassir / Чеки Т-Бизнеса
  - [ ] PAX + СБП на экране киоска (`tbank_pos_sbp`)
  - [ ] УМКА уже стоит (`legacy_umka`)

- [ ] Заявка в Т-Бизнес: эквайринг + устройство ([терминалы](https://www.tbank.ru/business/payments/acquiring/terminal/))

## aQsi (если вариант A)

- [ ] API-ключ aQsi (ЛК → Интеграции)
- [ ] Подтверждение **unattended** / автозаказ без кассира (письмо от api@aqsi.ru)
- [ ] Wi‑Fi IP терминала (для мониторинга)

## PAX + Smart Sale (если вариант B)

- [ ] Модель (AF6 / A8900)
- [ ] IP, порт 27015, SDK Windows
- [ ] Какая **онлайн-касса** Т-Банка будет (не УМКА?)

## СБП на киоске (если нужен крупный QR)

- [ ] TerminalKey + пароль Т‑Кассы
- [ ] Кто пробивает чек после СБП (aQsi / облачная касса)

## Опционально (legacy)

- [ ] УМКА и HS-K33 остаются на объекте? → `integration_mode: legacy_umka`
