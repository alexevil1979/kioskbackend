# NUC — компьютер киоска

## По ТЗ на объекте

- CPU: Intel i3
- RAM: 16 GB
- SSD: 120 GB
- ОС: Windows 10 Pro
- Экран: 32″, Full HD (1920×1080)

## Сетевые интерфейсы (использование)

| Интерфейс | Подключение | Назначение |
|-----------|-------------|------------|
| **Wi‑Fi** | Интернет-роутер | CRM, СБП, обновления, удалённая поддержка |
| **Ethernet** | Switch | УМКА, HS-K33, опционально POS |

## ПО киоска

- Python 3.11+, PyQt6
- Автозапуск: `scripts/install_autostart.ps1`
- Kiosk: полноэкранный режим + блокировка клавиш (`src/core/kiosk_win.py`)
- Логи: `logs/kiosk.log`

## Рекомендации Windows (kiosk hardening)

1. Отдельная учётная запись без прав администратора для автозапуска.
2. **Assigned Access** / Shell Launcher — только приложение киоска (опционально).
3. Ctrl+Alt+Del — ограничение через локальную политику (hook не гарантирует 100%).
4. Статический IP или DHCP reservation на LAN-интерфейсе NUC.
5. Отключить спящий режим и обновления в часы работы.

## Firewall

Разрешить исходящие: HTTPS (443) на CRM и `securepay.tinkoff.ru` / API Т‑Кассы.  
Разрешить входящие/исходящие в подсеть `192.168.1.0/24`: HTTP к кассе, TCP 27015 к POS, порт принтера.
