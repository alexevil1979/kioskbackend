# Сетевая топология (Т-Банк first)

## Минимальная схема (рекомендуется)

```
         ┌─────────────┐
         │   Роутер    │◄── интернет (CRM, aQsi API, Т‑Касса)
         └──────┬──────┘
                │ Wi‑Fi
     ┌──────────┴──────────┐
     │                     │
┌────▼────┐          ┌─────▼─────┐
│ NUC     │          │ aQsi 6    │
│ киоск   │          │ Т-Банк    │
│ PyQt6   │          │ оплата+ККТ│
└─────────┘          └───────────┘
```

**Switch, УМКА, HS-K33 не обязательны.**

## Расширенная схема (если POS по кабелю)

```
NUC (Wi‑Fi) ──► роутер ◄── Wi‑Fi/LAN ── POS PAX
NUC (LAN)   ──► switch ──► POS (Ethernet), опционально своя касса
```

## Каналы данных

| Задача | Канал |
|--------|--------|
| CRM, СБП (экран киоска) | NUC Wi‑Fi → интернет |
| aQsi API | NUC Wi‑Fi → `api.aqsi.ru` |
| Smart Sale (PAX) | NUC LAN/Wi‑Fi → `IP:27015` |
| УМКА (legacy) | NUC LAN → switch → касса |

## Пример IP (`hardware.*` в settings)

| Узел | Пример |
|------|--------|
| NUC | DHCP Wi‑Fi |
| aQsi 6 | 192.168.1.102 (резервация DHCP) |
| PAX (если есть) | 192.168.1.103 |
