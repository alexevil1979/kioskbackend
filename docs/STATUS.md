## Известные ограничения

| Тема | Описание |
|------|----------|
| Путь с кириллицей | Загрузка фото только через `image_utils.load_pixmap()` |
| Размер JPEG | Демо-фото ~2 МБ каждое — на слабом диске можно оптимизировать |
| Keyboard hook | В dev-среде часто ERROR в логе — на киоске с правами админа проверить отдельно |
| Платежи / фискализация | Все `use_mock: true` до ключей и API |
